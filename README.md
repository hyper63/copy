## Quick Start

``` sh
curl -O https://hyperland.s3.amazon.aws.com/hyper-copy
chmod +x hyper-export
hyper-copy [SRC_CONNECTION_STRING] [TARGET_CONNECTION_STRING]
```

* Linux x64 Environment
